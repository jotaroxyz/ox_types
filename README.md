# ox_types

Consolidated types for [Lua Language Server](https://github.com/LuaLS/lua-language-server) to use with our FiveM resources.

## Usage

- Install and setup [fivem-lls-addon](https://github.com/overextended/fivem-lls-addon).
- Add this repo to your lua addons directory.

```sh
git clone https://github.com/overextended/fivem-lls-addon.git
```
